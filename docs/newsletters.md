## How to Stay Current

1. **Subscribe to 3–5 newsletters** — more than that becomes overwhelming
2. **Follow company engineering blogs** via RSS (use Feedly or similar)
3. **Set up Google Alerts** for "data engineering", "Apache Iceberg", "dbt"
4. **Follow tool release notes** on GitHub (Watch → Releases Only)
5. **Join communities** — news breaks there first
